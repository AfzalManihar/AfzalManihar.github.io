# Apex AI Portfolio

A Next.js 15 portfolio for a Junior Data Scientist, ML Engineer, or AI Engineer.

## Run locally

```bash
npm install
npm run dev
```

## Personalize before launch

- Replace `Alex Morgan`, email, LinkedIn and GitHub URLs in `src/app/page.tsx`.
- Replace project URLs in `src/data/portfolio.ts`.
- Add your PDF as `public/resume.pdf`; the resume CTA is already wired to it.
- Replace the credential list and timeline with verified experience.

Project content is deliberately centralized in `src/data/portfolio.ts`; adding another case study requires only a data entry and its image in `public/projects`.
